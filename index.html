Frequently Asked Questions

Everything you need to know about No Mouse Mode

How do I control my Mac cursor without a mouse?

With No Mouse Mode, press Cmd+Shift+M to activate keyboard cursor control. Use WASD keys to move the cursor, Space to left-click, Return to right-click, and arrow keys to scroll. It's that simple.

What keys does No Mouse Mode use?

By default: W/A/S/D for cursor movement (up/left/down/right), Space for left-click, Return for right-click, Arrow keys for scrolling, and Shift for speed boost. All keys are fully customizable in settings.

Does it work with macOS Sonoma and Sequoia?

Yes! No Mouse Mode works with macOS 13.0 (Ventura) and all later versions including Sonoma and Sequoia. It runs natively on both Intel and Apple Silicon Macs.

Is this a subscription or one-time purchase?

No Mouse Mode is a one-time purchase of $6.99. No subscriptions, no recurring fees. You own it forever and get free updates included.

Can I customize the keyboard shortcuts?

Absolutely! Every key binding is customizable. You can remap cursor movement, click actions, scrolling, and speed boost to any keys you prefer. Adjust cursor speed, scroll speed, and acceleration too.

What's the difference from macOS Mouse Keys?

No Mouse Mode uses intuitive WASD controls (like gaming), offers smooth scrolling with arrow keys, includes a speed boost feature, and provides full customization — features not available in the built-in macOS Mouse Keys.
